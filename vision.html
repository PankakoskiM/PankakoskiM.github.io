Vision's technology

Labra AI VISION is a modern, deep learning enabled machine vision system for quality assurance, measurement and automation. VISION uses neural networks to learn distinctive class features from example objects which it is shown. The software trains a neural network on your behalf by comparing example images which the user sets to different classes.

During training the software automatically finds features from the images which separate the classes. These features are then used by the neural network to classify your production. Features can be differences in the shape of the object, differences in color, differences in patterns, cracks or other surface defects.

The software finds these features automatically - the user can train and run a neural network model from the Graphic User Interface (GUI) without having to write a line of code or specify classification rules. It is possible to start using the software with only a relatively small set of example data and increase accuracy over time with more examples.

Use cases

Quality Assurance

Identify and remove faulty produce from your production line with any distinguishing visual factor. You don't need to know what the features are or how to express them to a computer. Simply let Vision image your production and learn from the images it takes. If a human can see the error from an image, so can Vision.

Product Classification

Similarly to identifying faulty produce, Vision can classify which product it is imaging. You can use this information later for automation or data collection purposes.

Automation

Vision can track the speed, location and size of an object. This information can be relayed to your automation system for the removal of the item.

Error highlighting

Vision can highlight the area of the item where the potential error within the product is. This can be used to assist a human operator or used later to understand potential flaws in production.

Measurement

Vision can measure the length, width and height of your produce including the sizes of components or holes.

Data collection

Vision stores the images of the classified produce as well as creates statistics on the quality of your production.
